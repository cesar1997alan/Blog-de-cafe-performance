# Blog de Café (Optimizado)

Proyecto para aprender y aplicar optimizaciones de rendimiento, manipulación del DOM y buenas prácticas en desarrollo frontend.

Este es un **blog de café estático** creado con **HTML, CSS y JavaScript**, enfocado en mejorar la experiencia del usuario, validar un formulario, y aplicar conceptos de performance en un proyecto real.

## ¿Qué se practica en este proyecto?

Este proyecto te permite demostrar que sabes:

-  Estructurar un sitio web con HTML semántico.
-  Estilizar con CSS de forma **responsive**.
-  Manipular el DOM usando JavaScript.
-  Validar formularios y gestionar eventos de usuario.
-  Aplicar buenas prácticas para **performance** (optimización de carga, manejo eficiente del DOM, etc).
-  Separar correctamente HTML, CSS y JS.
